# Estadísticas Históricas de Enfermedades Pulmonares en España Antes y Después del COVID-19

Part of the Open Research Collection by Juan Moisés de la Serna Tuya — ORCID: 0000-0002-8401-8018

## Overview

Base de datos histórica sobre enfermedades pulmonares en España (2015-2023), organizada por año, comunidad autónoma y categoría de enfermedad. Incluye datos antes y después del COVID-19.

## Description

Categorías de enfermedades: EPOC (J44), Asma bronquial (J45-J46), Neumonía (J18), Cáncer de Pulmón (C34), Fibrosis Pulmonar Idiopática (J84), Enfermedades Respiratorias totales, EPOC + COVID-19.

Indicadores: Mortalidad, Hospitalizaciones, Prevalencia. Cobertura: 17 CCAA + Ceuta, Melilla + España. Período: 2015-2023 (Pre-COVID 2015-2019 | Durante-COVID 2020-2021 | Post-COVID 2022-2023).

Fuentes: INE, Ministerio de Sanidad (CMBD), SEPAR, EPISCAN I y II, GECP, APEPOC, JIACI.
